# E-commerce-Back-End

## Description
the back end for an e-commerce site, working Express.js API and configure it to use Sequelize to interact with a PostgreSQL database.

## Installation
- npm i<br>
- npm run start<br>

## Usage
- node index.js
- Insomnia GET, POST, PUT,and DELETE
